id.py

Python identicon generator

Short python script that uses the MD5 hash of an input string to generate a unique 5x5 identicon of a random colour, similar to those generated by Github. Allows the resultant icon to be saved to the system as a bitmap.

A complete open source version of Bethesda's Creation Kit built from the ground up in C++ with the Qt framework, with the intention of using Vulkan for 3D rendering. voxbox

Procedurally generated voxel engine

Developed as an educational exercise into graphics programming and OpenGL. Voxbox is a voxel engine that renders polygons based on a density function of a sparse octree, and supports multiple material types. A work in progress, the next step is to implement unlimited terrain and dual-contouring.
